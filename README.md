# Ecommerce

This is for the people who want to test this application.

## Setup Process

**Requirements** 
- pip
- venv 

**Commands**

```bash
python -m venv venv
source venv/bin/activate
cd kapda 
pip install -r requirements.txt
```
Now virtual environment is created and all the dependencies are installed.
